# Date The Map

A game in which it is the goal to guess the year portrayed by a historic map.
(not a dating sim)

## Installing
Go to [releases](https://github.com/FBLordn/DateTheMap/releases) and follow the instructions for your platform:

### Windows (64-bit)

Download and run the `.msi` or `.exe` file

### Linux

#### Debian

Download the `.deb` file and run `sudo apt install file_name.deb` (other package managers may also be used)

#### AppImage

Download and run the `.AppImage` file. You may need to make it executable (`chmod +x file_name.AppImage`)

### Other

Other platforms are not officially supported. Please open a ticket if you have a desire for one.
